## Example of "On this page"

```hbs

<DocfyOutput @fromCurrentURL={{true}} as |page|>
  {{#if page.headings.length}}
    On this page:
    <ul>
      {{#each page.headings as |heading|}}
        <li>
          <DocfyLink @to={{page.url}} @anchor={{heading.id}}>
            {{heading.title}}
          </DocfyLink>
        </li>
      {{/each}}
    </ul>
  {{/if}}
</DocfyOutput>
```
